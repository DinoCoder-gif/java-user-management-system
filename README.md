# Sistema de Gerenciamento de Usuarios (Java)
Projeto desenvolvido em Java com o objetivo de praticar lógica de programação, POO e estruturação de um CRUD em console.

# Funcionalidades
-Cadastrar usuário
-Listar usuários
-Buscar usuário por ID
-Atualizar usuário
-Remover usuário

# Tecnologias
-Java
-Scanner
-Collections (Arraylist)
-Git e GitHub

# Como executar
1- Clone o repositório
2- Abra em uma IDE (IntelliJ, Eclipse)
3- Execute a classe 'Main'
